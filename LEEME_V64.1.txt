OFERTAS IMPERDIBLES MX — V64.1
API CONSOLIDADA PARA VERCEL HOBBY

MOTIVO:
Vercel Hobby permite un máximo de 12 funciones Serverless por despliegue.
Cada archivo dentro de /api cuenta normalmente como una función.

REEMPLAZA:
- api/admin-publicidad.js
- admin/admin.js
- vercel.json

ELIMINA:
- api/admin-producto-mercadolibre.js
- api/actualizar-precios.js

IMPORTANTE:
Los dos archivos eliminados ya no son necesarios. Sus funciones quedaron
integradas en api/admin-publicidad.js mediante estas rutas:

- Consultar producto:
  /api/admin-publicidad?action=consultar-producto

- Actualizar precios:
  /api/admin-publicidad?action=actualizar-precios

RESULTADO:
- Se conservan la consulta de precio, imagen y título.
- Se conserva la selección automática del mejor cupón.
- Se conserva el botón manual “Actualizar precios”.
- Se conserva la actualización diaria programada.
- Se reducen dos funciones Serverless.

SQL:
No ejecutes ningún SQL para esta actualización.

PASOS:
1. Reemplaza los tres archivos indicados.
2. Borra físicamente del repositorio los dos archivos API indicados.
3. Confirma el cambio o haz commit en GitHub.
4. Espera el nuevo despliegue de Vercel.
